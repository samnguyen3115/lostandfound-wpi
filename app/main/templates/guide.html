How to Use the Lost and Found Web App

1. Posting a Lost Item

  1. Go to the "Report Item" dropdown in the navigation bar and select "Report Lost Item".
  2. Fill in the required details, such as a description of the item, where you lost it, and the reward you're offering (if any).
  3. Upload a clear image of the lost item to help others identify it more easily.
  4. Click Submit to post your lost item.

2. Posting a Found Item

  1. Go to the "Report Item" dropdown in the navigation bar and select "Report Found Item".
  2. Fill in the required details, such as a description of the item and where you found it.
  3. Upload a clear image of the found item.
  4. Click Submit to post the found item.

3. Using the Lost and Found Map

  1. Navigate to the "Lost and Found Map" page using the link in the navigation bar.
  2. On the map, click on the image of the building where you think the item might have been lost or found.
  3. The app will show you the location where lost and found items are usually kept within that building.

Tip: Check these locations before posting a lost or found item to save time!

4. Viewing Posts

  1. Log in and go to your Profile to view the items you have posted.
  2. Browse the dashboard to view items posted by others.
  3. Use filters like color or building location to narrow down the posts.

5. Claiming an Item

  1. If you find an item that matches your lost item, contact the poster using the contact information provided in the post.
  2. Arrange to meet and verify the item's details.

Note: Always verify the ownership of the item before handing it over.
